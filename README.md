# Todo App

A full-stack Todo application with **Django REST Framework** backend and **React + Vite** frontend. Includes authentication, authorization, and user management.

---

## Features

- User registration and login with JWT authentication
- User roles (Admin, Worker)
- Account activation and authorization checks
- CRUD operations for Todos
- Responsive frontend using React + Tailwind CSS
- Centralized error handling and token refresh
